# 小型进销存系统

### 功能要求

使用SSM(Spring MVC + Spring + MyBatis)实现一个小型进销存系统的部分功 能。MySQL 5.5作为后台数据库。共计5个主要功能

| 序号 | 功能                                   |
| :--: | :------------------------------------- |
|  1   | 用户登录                               |
|  2   | 退出登录                               |
|  3   | 销售                                   |
|  4   | 对销售记录进行分页查询并按指定规则排序 |
|  5   | 根据商品名称查询该商品的库存数量       |
